A [Flarum](https://flarum.org) extension. 

## Installation

Install with composer:

```sh
composer require wusong8899/flarum-email:"*"
```

## Updating

```sh
composer update wusong8899/flarum-email:"*"
php flarum migrate
php flarum cache:clear
```
